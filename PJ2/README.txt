Attention: To run the code, please paste the command to console rather than use the run button in your IDE

How to run feedforward network:
--------------------------------
1.For test, run(model will be loaded from './models/feedforward_model.pth'):

python feedforward.py --test_5_runs

2.For train, run(model will be saved as './models/feedforward_model.pth'):

python feedforward.py --train

How to run CNN:
---------------
1.For test, run(model will be loaded from './models/cnn_model.pth'):

python cnn.py --test_5_runs

2.For train, run(model will be saved as './models/cnn_model.pth'):

python cnn.py --train
